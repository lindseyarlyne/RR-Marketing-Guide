# Claude Workflow Guide — Ritual Runway

How to use Claude efficiently as a solo founder. One conversation = one task.

---

## The Golden Rule

**One conversation = one focused task. Start fresh when switching tasks.**

Long conversations cause Claude to contradict itself and blend contexts. Keep sessions short and focused.

---

## How to Use Plugins (Desktop App Only)

Plugins replace manual role prompts for common tasks. Activate per conversation via the + icon in the chat toolbar.

| Plugin | When to Use |
|--------|------------|
| **Marketing** | Instagram/Pinterest content, brand voice, campaign planning |
| **Design** | UI reviews, UX critique, accessibility checks |
| **Productivity** | Daily task planning, launch tracking |
| **Engineering** | Code review, architecture decisions (alongside Cursor) |
| **Finance** | Revenue tracking once Ritual Runway has paying users |

---

## When to Use a Custom Role Prompt

Use a role prompt (from the Role Guide) only for roles not covered by plugins:

- Consultant + Creative Director
- Customer Support
- Growth Strategist
- Launch Strategist
- Copywriter (if you want tighter brand voice control)

Paste the role prompt as your **first message** in a new conversation.

---

## Conversation Types

| Type | What to Do |
|------|-----------|
| Dev work (Cursor prompts) | New conversation. Paste system context if needed. |
| Copy / content | New conversation. Activate Marketing plugin OR paste Copywriter role. |
| Strategy / decisions | New conversation. Activate relevant plugin OR paste role prompt. |
| Quick question | Any conversation, keep it short. |

## Memory Tips

- Claude builds memories from conversations over time.
- Say "remember that..." to explicitly store something.
- Review + edit memories: Settings → Memory.
- Project system prompt is your most reliable persistent context.
