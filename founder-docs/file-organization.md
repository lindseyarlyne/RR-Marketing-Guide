# File Organization — iCloud + Notion

Simple structure for a solo founder in launch prep.

---

## iCloud Folder Structure

```
Ritual Runway/
├── Brand/
│   ├── Logos + icons
│   ├── Color palette + fonts
│   └── Brand guidelines (if exists)
├── Design/
│   ├── Screenshots + mockups
│   ├── App store assets (future)
│   └── Canva exports
├── Product/
│   ├── Feature specs
│   ├── Bug notes
│   └── Cursor prompt notes
├── Business/
│   ├── Revenue + pricing notes
│   ├── Stripe setup notes
│   └── Launch plans
└── Etsy/
    ├── Worksheet templates (Canva exports)
    └── Listing copy
```

---

## Notion (Living Docs)

Use Notion for anything you update regularly:

- **Roadmap + Parking Lot** — active features, deferred items
- **Decisions Log** — what was decided and why
- **Launch Checklist** — phase-by-phase launch steps
- **Outreach Tracker** — which channels done, results

---

## GitHub (Code Only)

- `ritual-runway` repo — all app code, managed by Cursor
- `RR-Marketing-Guide` repo — this folder of founder reference docs

**Do not put code in the marketing repo. Do not put docs in the code repo.**

## Claude Project

Upload key reference files directly into your Ritual Runway Claude Project so Claude can reference them mid-conversation without you copy-pasting:
