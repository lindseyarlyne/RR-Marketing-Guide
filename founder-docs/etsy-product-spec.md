# Etsy Product Spec — Paycheck Budget Worksheet

**Product:** Ritual Runway Paycheck Budget Worksheet  
**Format:** Editable Canva Template — Light + Dark versions  
**Price:** $14  

---

## What's Included

5 pages × 2 colorways = 10 files total.

### Page 1 — Paycheck Allocation
The hero page.
- Pay date, pay period dates, total income at top
- Four categories: Fixed Bills, Variable Spending, Savings, Left Over (runway)
- Total spoken for at bottom

### Page 2 — Bill Due Dates Tracker
- Two columns: Pay Period 1 and Pay Period 2
- Each column: bill name, due date, amount, checkbox
- Built for biweekly earners who need to know which bills come out of which paycheck

### Page 3 — Monthly Overview
- Side-by-side snapshot of both paychecks for the month
- Income, total spent, total saved, runway remaining

### Page 4 — Notes / Intentions
- Minimal lined page
- Header: "What this paycheck is for" (not "goals" — Ritual Runway voice)

---

## Design Direction

**Dark version:** Near-black background, cream/off-white text, serif headings, monospace for numbers. Ritual Runway brand.

**Light version:** Warm linen/cream background, dark charcoal text, same typography. Broader appeal, Pinterest-friendly.

Both versions: identical layout, different colorway only.

---

## Delivery

- 2 Canva template links (light + dark), fully editable
- 2 print-ready PDFs exported at 300dpi (bonus files)

---

## Build Order in Canva

1. Build dark version first (on-brand)
2. Duplicate entire design, swap colorway
3. Export both as PDF Print (300dpi) for printable versions
4. Share both as Canva template links for editable versions
