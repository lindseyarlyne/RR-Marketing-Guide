# Claude Desktop Plugins — Reference

Built by Anthropic. Desktop app only (not available on mobile or web).

Each plugin = its own contained context. No memory bleed between roles.

---

## How to Activate

1. Open Claude Desktop app
2. Start a new conversation
3. Click **+** in the chat toolbar
4. Select **Plugins**
5. Activate the plugin for that task

---
