08
The "Ritual Runway" lockup
Brand signature
Ritual Runway
"Ritual" is always mauve. "Runway" is always ink (or pearl white on dark). One italic line, Cormorant Garamond, single space between words. Never stack vertically except in tight square spaces (avatars, app icons).
09
Quick Rules
Print this side
✓ Do
Use Cormorant italic for anything emotional or branded (headlines, "Ritual", taglines)
✗ Don't
Mix Cormorant with another serif. One serif only.
✓ Do
Use IBM Plex Mono for all dollar amounts, dates, labels, and small caps
✗ Don't
Use Plex Mono for body paragraphs — too dense to read.
✓ Do
Stick to the mauve family. If you need a second color, dim or lighten an existing mauve.
✗ Don't
Introduce green, gold, pink, blue, or any new hue. Mauve family only.
✓ Do
Round all rectangles to 12px. Round all buttons to 999px (full pill).
✗ Don't
Use sharp corners or 4px micro-rounded boxes. Inconsistent radii break the feel.
✓ Do
Leave generous whitespace. 24–32px padding inside cards. Lots of air.
✗ Don't
Pack content edge-to-edge. The brand is "calm + intentional" — let it breathe.
✓ Do
Use lowercase letterspacing 0.16–0.18em on UPPERCASE Plex Mono labels.
✗ Don't
Use ALL CAPS in any other font. Only Plex Mono does uppercase.
10
Canva-Specific Tips
Save these as a Brand Kit
Set up your Brand Kit. In Canva, go to Brand → Brand Kit → add all 12 hex codes above as Brand Colors, then add the 3 fonts. Every new design will auto-suggest them.
Save logo as Brand Asset. Upload SVG (light + dark) to Brand → Logos. They'll appear in every design's sidebar for one-click insert.
Italic in Canva. For Cormorant italic: select text → pick "Cormorant Garamond" → click the italic I button in the top bar. Don't just type in italics — it won't render correctly.
Letter spacing for labels. Plex Mono uppercase labels need letter spacing +180 (Canva's slider value). Set this once, save as a Text Style.
Templates to build first. Instagram square (1080×1080), Instagram story (1080×1920), Pinterest pin (1000×1500), email header (600×200). All on Parchment #FDFAF6 background.
Avoid Canva default rounding. When you add a rectangle, manually set corner radius to 12. Default 0 (sharp) and 100 (super round) both look off-brand.
